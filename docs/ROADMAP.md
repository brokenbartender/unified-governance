# Roadmap

## Phase 1: MVP (2-4 weeks)
- Core API: policies, resources, evaluations
- SQLite-based persistence
- Policy engine v0 (allow/deny)
- Basic evidence logs

## Phase 2: Evidence Packs (4-6 weeks)
- Exportable audit bundles (JSON/CSV)
- Signed evaluation logs
- Framework mapping templates (SOC2, ISO)

## Phase 3: Connectors (6-10 weeks)
- QuickBooks/Xero-style connector SDK pattern
- First 2 connectors (e.g., Snowflake + Google Drive)
- Event ingestion webhook

## Phase 4: Enterprise Readiness (10-14 weeks)
- Multi-tenant separation
- SSO (SAML/OIDC)
- RBAC for policies
